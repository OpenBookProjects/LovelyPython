== threading ==

多线程实做:
{{{#!python
import threading

def worker():
    """thread worker function"""
    print 'Worker'
    return

threads = []
for i in range(5):
    t = threading.Thread(target=worker)
    threads.append(t)
    t.start()
}}}

输出5次Worker
{{{
$ python threading_simple.py
Worker
Worker
Worker
Worker
Worker
}}}
